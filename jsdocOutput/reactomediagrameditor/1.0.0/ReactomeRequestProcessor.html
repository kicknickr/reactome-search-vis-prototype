ReactomeRequestProcessor

ReactomeRequestProcessor

Processes requests for information about the Reactome knowledge base. Leverages the Reactome API ( see ReactomeAPIQuerier ). Performs data transformations so that way UI logic can access data in a more convenient format. Creates internal data-structures and lookup tables to speed-up subsequent/related requests. // * @property {ReactomeAPIQuerier} reactomeAPIQuerier - // * The instance of the ReactomeAPIQuerier (see ReactomeAPIQuerier) // * It is the only source of information about the Reactome knowledge base.
