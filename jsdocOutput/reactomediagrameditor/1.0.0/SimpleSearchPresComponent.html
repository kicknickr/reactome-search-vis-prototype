SimpleSearchPresComponent

SimpleSearchPresComponent

A component for a "simple search" in which there is a text field for entering a query string and results displayed below it. The class is expected to be used by BaseSearchComponent (or a specific impl) and also possibly PaneComponent. The results are supplied externally with the SimpleSearchPresComponent.addResult method. The query that produces these results may gather query parameters from other UI components and is not limited to this one query string.
